Partie 1 :

Question biologique :

Comment la rsilience des rcifs coralliens, soit la capacit ne pas seffondrer suite a un stresse ou une perturbation varie-t-elle selon leffort de pche, limpact des ouragans et la rugosit initiale?

Hypothse :

  1. La rsilience diminue quand leffort de pche augmente.
  2. La rsilience diminue quand les ouragans sont plus frquence.
  3. La rsilience diminue quand les ouragans sont plus puissants.
  4. La rsilience diminue quand la rugosit est faible.

Mthode :

Modle de rugosit fixe:

\(\frac{dC}{dt} = \small{rTC - dC - aMC}\)
\(\frac{dM}{dt} = \small{aMC} - \frac{\alpha \frac{P}{\beta}M}{M + Y} - \small{\gamma MY}\)
\(\frac{dP}{dt} = \small{sP(1 -} \frac{P}{\beta K(M+T,R)} \small{ - sfP}\)
\(\frac{dY}{dt} = \small{1 - M - C}\)

La capacit de support de lenvironnement pour le poisson perroquet prend en compte labondance des algues, qui constituent leur nourriture, ainsi que la rugosit, qui constitue de lhabitat K(M + T, R).
\(\small{K(M+T,R) = }\frac{(c1R + c2)}{K_{max}}\frac{\delta(M+Y)}{1 + v(M+Y)}\)

Modle Nondimentionner par rapport aux taux de croissance des poisson perroquets par anne avec rugosit fixe:
\(\frac{dC}{dts} = (\small{rTC - dC - aMC})\frac{1}{s}\)
\(\frac{dM}{dts} = (\small{aMC} - \frac{\alpha \frac{P}{\beta}M}{M + Y} - \small{\gamma MY})\frac{1}{s}\)
\(\frac{dP}{dts} = \small{P(1 -} \frac{P}{\beta K(M+T,R)} \small{ - fP}\)
\(\frac{dY}{dts} = \small{1 - M - C}\)

Modle avec une rugosit variable:
\(\frac{dC}{dt} = \small{rTC - dC - aMC}\)
\(\frac{dM}{dt} = \small{aMC} - \frac{\alpha \frac{P}{\beta}M}{M + Y} - \small{\gamma MY}\)
\(\frac{dP}{dt} = \small{sP(1 -} \frac{P}{\beta K(M+T,R)} \small{ - sfP}\)
\(\frac{dR}{dt} = \small{h_GC(3-R) - h_E(1-C)(R-1)}\)
\(\frac{dY}{dt} = \small{1 - M - C}\)

Mise en uvre dun ouragan stochastique:
\(\small{dC = g_C(C,M,P,R,t)dt - C(F() + \sigma \epsilon)dq}\)
\(\small{dM = g_M(C,M,P,R,t)dt - Mm_Hdq}\)
\(\small{dP = g_P(C,M,P,R,t)dt}\)
\(\small{dR = g_R(C,M,P,R,t)dt - (R-1)(G() + \sigma \epsilon)dq}\)
\(\small{dY = 1 - M - C}\)

\(\small{F() = 0.0512 + 0.0176*te}\) avec un maximum de F() = 0.5. G() est soit 0.5 pour un impact lever, F() pour un impact moyen et 0.05 pour un impact faible.

o \(\small{g_C(C,M,P,R,t),g_M(C,M,P,R,t),g_P(C,M,P,R,t),g_R(C,M,P,R,t)}\) sont des fonctions qui represente le modle avec avec un rugosit variable.
Et o q est un processus de saut de Poisson avec un taux de saut tel que dq = 0 avec une probabilit de 1 - dt et dq = 1 avec une probabilit de dt.

Supposition :

  1. Aucun changement dans la temprature, la chimie de leau ou llvation du niveau de la mer nest significatif sur lchelle du temps tudier.
  2. Les perturbations ou stress des rcifs coralliens autres que les ouragans, les impacts indirects de la pche ou la comptition de lespace avec les macroalgues ne sont pas pris en compte.
  3. Leffort de pche est constant pour toutes les annes de la simulation.
  4. Aprs un ouragan, le gazon dalgues colonise immdiatement tout lespace libre.
  5. Le gazon dalgue correspond toute la surface qui nest pas des coraux ou des macroalgues.
  6. Les ouragans nont pas deffets directs significatifs sur labondance ou la croissance des poisson perroquets.
  7. Les macroalgues ont le mme niveau de dommage, peu importe le niveau dimpact des ouragans.
  8. Les ouragans ont un impact et une frquence constante sur toute la dure de la simulation.
  9. Les effets des ouragans sont immdiats sur les populations de lcosystme.
  10. La distribution des coraux, des algues macroscopiques et des poissons-perroquets dans lespace na pas dimpact sur leur dynamique.
  11. Un ouragan affecte lensemble de lcosystme de faon gale.
  12. Les ouragans suit un processus de saut de Poisson avec un taux de saut tel que dq = 0 avec une probabilit de 1 - dt et dq = 1 avec une probabilit de dt.
  13. Tous les paramtres du modle sont reprsentatifs de la situation relle de lcosystme.
  14. Il est impossible pour le corail et les macroalgues de steindre totalement.

# Fonction pour le modele nondimentionner par rapport taux de croissance des poisson perroquet par anne avec rugosit fixe : 
interaction_coral <- function(t, vars, parms){
  with(as.list(c(parms, vars)), {
    
    Y <- 1 - M - C # dY/dt
    R <- rug
    
    dM <- (a*M*C - P*M/(M+Y) + gamma*M*Y)/s # dM/dt
    dC <- (r*Y*C - d*C - a*M*C)/s  # dC/dt
    dP <- P*(1 - (P/(((ca+cb*R)/kmax)*((delta*(M+Y))/(1+v*(M+Y)))))) - (f)*P # dP/dt
    
    # Rsultat
    res <- c(dM=dM, dC=dC, dP=dP)
    return(list(res))
  })
}

Pour construire les graphiques du couvert de corail final moyen et de la rugosit finale moyenne selon la rugosit initiale et leffort de pche, nous avons utilis la fonction suivante. Elle inclut lincidence alatoire douragan avec diffrentes gravits dimpacts dfinis, et occupant diffrentes frquences moyennes. Les catgories dimpacts sont faibles (1), modres (2) et importantes (3) et les diffrentes frquences moyennes sont 1 fois par 9 ans (1/9), une fois par 15 ans (1/15) et une fois par 30 ans (1/30).

corailstochastique <- function (R0,f,lambda=1/30,impact=2,dt=1,graph=FALSE){
  #on instaure une tendue de temps
  tmax <- 100
  t <- 0
  #le temps au dernier ouragan
  to <- 0
  
  #on pose les valeurs de dpart et les paramtres
  C <- 0.56
  M <- 0.01
  Y <- 0.43
  R <- R0
  P <- 0.5
  a<-0.1
  g<-0.8
  d<-0.44
  r<-1
  f<-f
  s<-0.49
  ca<--(3.21)
  cb<-3.65
  kmax<-17.745
  delta<-4.557
  v<-0.9877
  hG<-0.03
  hE<-0.01
  o <- 0.05
  mH <- 0.8

  #on initie les vecteurs de rsultats
  vecC <- C
  vecM <- M
  vecY <- Y
  vecR <- R
  vecP <- P
  
  while (t<tmax){
    
    #on effectue le tirage pour dterminer si il y a ou pas ouragan
    ouragan <- pmin(rpois(1,lambda*dt),1)
    #si oui ou non, on ajuste dq
    if (ouragan==1){
      
      #on calcule le temps depuis le dernier ouragan
      te <- t - to
      
      #on sauve le nouveau temps de dernier ouragan
      to <- t
      
      #on calcule F
      F <- pmin(0.0512 + 0.0176*te, 0.5)
      
      #diffrents dommage selon valeur d'impact
      if (impact==3){
        #on calcule les dommages d'un fort ouragan
        dommC <- C*(F + o*runif(1))
        dommM <- M*mH
        dommR <- (R-1)*0.5
      }
      if (impact==2){
        #on calcule les dommages d'un moyen ouragan
        dommC <- C*(F + o*runif(1))
        dommM <- M*mH
        dommR <- (R-1)*(F + o*runif(1))
      }
      if (impact==1){
        #on calcule les dommages d'un faible ouragan
        dommC <- 0
        dommM <- M*mH
        dommR <- (R-1)*0.05
      }
    }else{
      dommC <- 0
      dommM <- 0
      dommR <- 0
    }
    
    #on calcule les changements
    dM <- ((a)*M*C - ((1)*P*M/(M+Y)) + (g)*M*Y)*dt - dommM
    dC <- ((r)*Y*C - (d)*C - (a)*M*C)*dt - dommC
    dP <- ((s)*P*(1 - (P/((((ca+cb*R)/kmax)*((delta*(M+Y))/(1+v*(M+Y))))))) -(f*s)*P)*dt 
    dR <- ((hG)*C*(3 - R) - (hE)*(1 - C)*(R - 1))*dt - dommR
    
    #on effectue les changements
    M <- pmax(M + dM,0.0001)
    C <- pmax(C + dC,0.0001)
    Y <- 1 - M - C
    R <- R + dR
    P <- pmax(P + dP, 0.0001)
    
    #on sauve les nouvelles valeurs dans leur vecteurs
    vecC <- c(vecC,C)
    vecM <- c(vecM,M)
    vecY <- c(vecY,Y)
    vecR <- c(vecR,R)
    vecP <- c(vecP,P)
    
    #on fait avancer le temps
    t <- t + dt
  }
  #si on veut afficher les graphiques, on met TRUE  l'argument graph = 
  if (graph==TRUE){
    #on affiche le premier plot, avec la trajectoire du corail
    plot(seq(0,tmax,length.out = length(vecC)),vecC, type="l", col="red", ylim=c(0,1), 
         xlab= "temps", ylab="valeurs")
    #on ajoute un axe secondaire pour la rugosit
    axis(side = 4,at= seq(0,1,0.1),labels = seq(1,3,0.2), )
    mtext("rugosit", side = 4, line = 2, col = "black", cex = 1)
    #on affiche les diffrentes lignes
    lines(seq(0,tmax,length.out = length(vecM)),vecM, col="blue")
    lines(seq(0,tmax,length.out = length(vecP)),vecP, col="green")
    lines(seq(0,tmax,length.out = length(vecR)),(vecR-1)/2, col="orange")
    #on ajoute une lgende
    legend("topright", legend = c("corail", "macroalgues","poissons","rugosit"),
           col = c("red","blue","green","orange"), lty = c(1, 1),
           lwd = 1.75, bg = "white",cex = 0.65)
  }
  #on affiche le rsultat final du couvert de corail ainsi que de la rugosit
  return(c(tail(vecC,1),tail(vecR,1)))
}

graph.imageC <- function(impact,freq,it=10,pas=0.1){

  matC <- matrix(0, nrow = 21,ncol = 26)
  matsd <- matrix(0, nrow = 21,ncol = 26)

  #on fixe les variables constantes
  imp<-impact
  l<-freq
  #Les boucles suivantes font essayer diffrentes valeurs de R0 et de f
  for (i in 1:21) {
    for (j in 1:26){
      #on amorce un vecteur pour contenir les rsultats de chaque rplicat
